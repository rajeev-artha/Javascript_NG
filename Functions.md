# Functions

Aapne isse pehle built-in methods aur functions dekhe ( jaise parseInt() ) jiska use karke aap kuch kaam kar sakte hain.

Aap khud se bhi functions define kar sakte hain. Inhe hum user-defined funcitons kehte hain.

Example:
```javascript
//defining the function "add" here
> function add (number1, number2) {
  var result = number1 + number2;
  return result;
}

//using the funciton
> var sum = add (2,3);
> sum
//expected output 5.
 ```
 
 Is example mein hum pehle *add* function ko define kar rahe hain. *number1* aur *number2* is function ke arguments hain. *add* functions in dono numbers ko + kar deta hai aur result return karta hai.
 Function ko use karne ke liye hum function *add* ko call karte hain aur use 2 and 3 arguments pass karte hain.
 
 Function definition ke baare mein aur [yahan](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Functions) padh sakte hain.
